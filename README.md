# 🔐 Secure Password Generator

A sleek, intuitive, and modern web application that generates highly secure passwords on the fly. Created with clean HTML, CSS, and Vanilla JavaScript with no external dependencies.

## ✨ Features

- **Dark Glassmorphic UI**: Beautiful gradients and translucent panels providing a premium feel.
- **Bold & Aesthetic Typography**: Carefully chosen font weights and letter-spacing for ultimate readability and style.
- **Real-Time Generation**: Instantly creates passwords varying from 8 to 64 characters.
- **Dynamic Strength Indicator**: Visual feedback on the password's robustness (Weak / Medium / Strong) using a smoothly animated multi-segmented bar.
- **Customizable Criteria**: Granular options to include or exclude:
  - Uppercase letters (`A-Z`)
  - Lowercase letters (`a-z`)
  - Numbers (`0-9`)
  - Symbols (`!@#$%^...`)
- **One-Click Copy**: Seamlessly copy your newly minted password to the clipboard with minimal friction.
- **Fluid Responsiveness**: Adapts perfectly to mobile and desktop screens alike.

## 🚀 Getting Started

No configuration or package installations required! The application is fully self-contained.

1. **Clone or Download** the repository.
2. **Open `index.html`** in your favorite browser (Chrome, Firefox, Safari, Edge).
3. **Customize your parameters** and instantly get a secured password ready for use.

## 🛠️ Built With

- **HTML5**: Clean, Semantic web structure.
- **CSS3**: Custom properties (Variables), Flexbox, CSS Gradients, Glassmorphism via `backdrop-filter`, and engaging transition animations.
- **Vanilla JavaScript**: Lightweight DOM manipulation, simple random password generation logic, and secure Clipboard API capability.

## 📝 Usage

- Adjust the **Character Length** slider to decide how robust your password will be.
- Toggle between uppercase, lowercase, numbers, and symbols—an elegant warning will prevent generations if zero boxes are checked!
- Click **GENERATE**, then rely on the robust strength indicator to help inform you of the generated password's quality.
